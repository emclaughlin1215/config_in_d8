Assumptions

  1. Drupal 8
  2. Vanilla config
  3. Not working directly on dev/master branches
  4. Familiarity with git
  1. Will this workflow work in D9? Who knows!
  2. No additional modules like config-ignore.
  3. Please don't make me explain why.
  4. Command line/sourcetree, demo being on commandline

Basic Workflow

  1. Pull down the DB
  2. Pull down your master branch
  3. Run drush cim
  4. ~make your changes~
  5. Run drush cex
  6. ~commit only your changes~
  7. Merge changes into your master branch
  8. Push your master branch back to the remote
